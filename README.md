# ble_peripheral_soil_moisture

## SDKと開発環境の構築
本コードはnRF5 SDK v16.0.0を使用しています。
本コードをビルドする場合は、nRF5 SDK v16.0.0をダウンロードして、各ファイルとリンクしてください。

統合開発環境はSegger Embedded Studioを使用しています。

開発環境の構築方法およびSDKのダウンロード方法は以下の記事を参照してください。
